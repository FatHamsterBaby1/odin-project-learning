Style Tag

The style tag is used to define style information for a document. The style tag displays the appearance and different characteristics of a website or page.
Examples of this are font styles, colors, and spacing.

Link Tag

The Link tag allows you to click and interact with an external resource within your document or webpage. The link tag is used to bring users to another page that is filled with content of some kind.
These links can bring a user to different things such as videos, images, and, even other websites.

Base Tag

The Base tag is used to specifiy a base URL for links. This tag specifies the "base" or main URL to use for all URLs in a document. Note that for every document there can only be on base element.
